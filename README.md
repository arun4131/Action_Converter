# Action Converter 🚀

## 📌 Description
Action Converter is a simple AI-powered tool that helps users convert learning goals into clear, actionable steps. It solves the Curiosity-to-Action Gap by turning passive learning into structured execution.

## 🎯 Problem
Users often consume educational content but struggle to take the first step toward building skills.

## 💡 Solution
This tool takes a learning goal and available time per day, and generates a short, practical 5-day action plan.

## ⚙️ Features
- Input: Learning goal + time per day
- Output: 5-day actionable plan
- Simple and user-friendly interface

## 🛠️ Tech Stack
- Backend: Python (Flask)
- Frontend: HTML, Tailwind CSS
- AI: Groq API

## 🚀 How to Run

### Backend
```bash
python3 app.py
```

### Frontend
```bash
cd frontend
python3 -m http.server 5500
```

Open in browser:
http://localhost:5500

## 🔐 Security
API keys are managed using environment variables and are not exposed in the code.
